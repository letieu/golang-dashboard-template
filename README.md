# Dashboard app template
Starter point for build a fullstack dashboard app use:

- golang
- gin
- templ
- html

## Run
```sh
# run
templ generate -watch -cmd "go run main.go"

# compile tailwindcss
tailwindcss -i views/styles/input.css -o static/main.css -w
```
